## Что такое алгоритм?

**Алгоритм** — это точная конечная последовательность действий, направленная на достижение определённой цели или решение конкретной задачи.

### Основные свойства алгоритма:

1. **Дискретность** — алгоритм состоит из отдельных шагов
2. **Детерминированность** — каждый шаг однозначно определён
3. **Конечность** — алгоритм завершается за конечное число шагов
4. **Массовость** — применим для целого класса задач
5. **Результативность** — всегда приводит к результату

## Что такое блок-схема?

**Блок-схема** — это графическое представление алгоритма, в котором каждый шаг изображается в виде геометрических фигур, соединённых стрелками.
